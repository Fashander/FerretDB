# `$nor`

import CodeBlock from '@theme/CodeBlock';
import Dataset from '!!raw-loader!./logical-operators/2-insert-dataset.request.js';

The `$nor` operator combines multiple query expressions and returns all documents that do not match any of the specified query expressions.

The operator accepts an array of query expressions as shown below, and no expression must be satisfied for a document to be included in the result set.

```js
db.runCommand({
  find: '<collection_name>',
  filter: {
    $nor: [
      { <query-expression1> },
      { <query-expression2> }
    ]
  }
})
```

A query expression can be any valid query such as `{ <field>: <value> }`, `{ <field>: { <operator>: <value> } }`, `{ <field>: /<pattern>/ }`, etc.
